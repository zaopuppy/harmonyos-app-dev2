# ArkTS 语法参考

提供 ArkTS 语言的基础语法、类型系统、并发模型等参考。

## 基础类型

- 常用类型：number, string, boolean, void, null, undefined
- 对象类型：class, interface, enum, struct, type
- 集合类型：Array, Set, Map, Tuple

## 语法特性

- 装饰器 (@State, @Prop, @Link 等)
- 异步编程 (async/await, Promise)
- 泛型
- 模块导入导出

## 最佳实践

- 空安全处理
- 性能优化
- 代码风格规范
