# HarmonyOS App Dev Skills

HarmonyOS 应用开发技能工具箱，为 Agent 系统提供 ArkTS 语法、ArkUI 组件、UT 测试等开发辅助。

## Skills

| Skill | 用途 |
|-------|------|
| `harmony-arkts` | ArkTS 语法参考 |
| `harmony-arkui` | ArkUI 组件语法 |
| `harmony-template` | 应用/模块模板 |
| `harmony-ut` | 单元测试 |
| `harmony-cli` | 命令行工具 |

## 使用方式

Agent 可直接读取 `skills/<name>/skill.md` 获取对应技能指导。
